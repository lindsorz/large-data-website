Bioluminescence

The natural light emitted from living organisms

Bioluminescence isn’t something that’s understood today by the general public as easily as something like how water moves in and out of its three states (ice, water, steam). A website should be created that’s easily accessible to the public that is loaded with facts about the awesomeness that is bioluminescence. It really became a big interest when James Cameron used it in his movie Avatar.

What is really exciting about bioluminescence is the fact that it’s not restricted to one type of organism. There are many different types of fish, fungi, insects and plants that use bioluminescence in different ways. It’s also really beautiful to look at.

Purpose

The purpose of this project is to make a large data-orientated website. A website that has a very large amount of facts and information that’s organized, easy to navigate, easy to read, and is esthetically appealing and well designed.

What I would like to learn while making this website is to get a better understanding of how CSS works. I can understand the general basics of CSS but I always feel like there’s something missing. So I need to pay better attention to the websites I visit to see how they work. I also want to learn different ways of making the website interactive with a combination of CSS and Javascript.
